# SMS Spam Detection

## Project Overview
This project uses Machine Learning to classify SMS messages as Spam or Ham (Normal messages).

## Dataset
Dataset used: SMS Spam Collection Dataset

The dataset contains SMS messages with two categories:
- Ham: Normal messages
- Spam: Unwanted messages

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NLP (TF-IDF)

## Machine Learning Model
Multinomial Naive Bayes classifier is used for spam classification.

## Steps Performed
1. Loaded SMS dataset
2. Data preprocessing
3. Converted text messages into numerical features using TF-IDF
4. Split data into training and testing sets
5. Trained Naive Bayes model
6. Evaluated model accuracy
7. Predicted spam messages
